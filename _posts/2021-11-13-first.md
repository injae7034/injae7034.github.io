## 블로그 만들 때 참고

테디노트님 유튜브 보고 블로그 만들었어요.

덕분에 손쉽게 블로그 만들 수 있었습니다.

감사합니다^^ 

### C++코딩

```c++
#include<iostream>
using namespace std;

int main(int argc, char* argv[])
{
        cout << "테디노트님 유튜브 보고 블로그 만들었어요." << endl;
	cout << "덕분에 손쉽게 블로그 만들 수 있었습니다." << endl;
	cout << "감사합니다^^ " << endl;
}
```
